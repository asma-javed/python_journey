# Python Journey

This repository contains my Python learning exercises and projects.  
It is organized by days and topics to track my progress from beginner to advanced Python.

## 📚 Contents

- **Day 1 – Python Basics**
  - Variables and Data Types
  - Input / Output
  - If-Else Conditions
  - Loops (for and while)
  - Number Guessing Game with score counter

- **Future Days**
  - Lists, Tuples, Dictionaries
  - Functions and Mini Projects
  - Object-Oriented Programming (OOP)
  - Advanced Python Concepts

## ⚡ How to Run

1. Open the `.ipynb` files in [Jupyter Notebook](https://jupyter.org/) or [Google Colab](https://colab.research.google.com/).  
2. Run each cell sequentially to see outputs and interact with programs.  

## 🔧 Tools Used

- Python 3
- Jupyter Notebook / Google Colab
