## 📚 Day 2 – Lists, Dictionaries, and Functions

- **Lists**: storing multiple values, adding/removing items
- **Tuples**: immutable collections
- **Dictionaries**: key-value pairs for storing structured data
- **Functions**: reusable blocks of code
- Mini Project: Student Info System (average grade calculation)

**Exercise Highlights:**
- Manipulating lists and tuples
- Creating and updating dictionaries
- Writing functions with parameters and return values
- Combining data structures and functions in a mini project

## ⚡ How to Run

1. Open the `.ipynb` files in [Jupyter Notebook](https://jupyter.org/) or [Google Colab](https://colab.research.google.com/).  
2. Run each cell sequentially to see outputs and interact with programs.  
3. Make sure to save any changes you make in the notebook before committing to GitHub.
